variables
    is the name of a memory location that stores some data
    RULES
        a. variables are case sensitive
        b. 1st character is alphabet or _
        c. no comma or blankspace
        d. no other symbol than _

    data types is the kind of data u are storing in the variable
        char
        int
        float
        double

constants
    values that dont change (fixed)
    TYPES
        integer constants
            1, 2, 3, 0, -1, -2
        real constants
            1.0, 2.0, 3.14
        character constants
            'a', 'A', '#'

    to print integer
    printf("age is %d", age)

    to print character
    printf("character is %c", char)

    to print float
    printf("pi is %f", float)
